# Мои игры на Python

Добро пожаловать в мой сборник игр, написанных на Python!

## Игры в сборнике

### 🎮 Тетрис
Классическая игра, где нужно собирать линии из падающих фигур.
- Управление: ← → ↓ ↑, Пробел — мгновенное падение
- Написано с использованием pygame-ce

## Как запустить
1. Установи Python 3.14+
2. Установи библиотеку: `pip install pygame-ce`
3. Запусти любую игру: `python tetris.py`
